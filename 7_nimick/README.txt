t_query: The Rust MBTA server
--------
Cargo.lock - autogenerated dependency information for Cargo
Cargo.toml - Cargo package information
data/connections.dat - a list of connections between lines
data/green.dat - Green line T stations
data/blue.dat - Blue line T stations
data/red.dat - Red line T stations
data/orange.dat - Orange line T stations
src/main.rs - main program code
src/graph.rs - our basic graph data structure
src/query.rs - our query handling module
src/print.rs - our module for outputting to the user
src/t.rs - our t datastructure module

example_graph.png - An image depicting how the T graph is represented
